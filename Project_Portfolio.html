DocuTuck – AI PDF Chatbot

Artificial Intelligence • Document Intelligence • Award Winning Project

DocuTuck was created to solve the challenge of reading and searching extremely large PDF documents that can span hundreds or even thousands of pages, such as textbooks, legal documents, research papers, and technical manuals. Manually scrolling and searching through these documents is inefficient and time-consuming.

Users simply upload a PDF and ask questions in natural language. The system intelligently retrieves relevant sections using vector embeddings and generates summarized answers instantly. This removes the need to manually copy content into external AI tools for summarization.

The project won First Prize in the Department Project Expo (2023–2024) for its practical usefulness and strong technical implementation.

Smart Stump Vision – AI for Cricket Decisions

Computer Vision • Deep Learning • Real-Time Analytics

Smart Stump Vision is an AI-powered computer vision system designed to support cricket umpiring decisions, especially in close situations such as run-outs and stumpings. The system detects critical objects including the bat, foot, and ground using a trained YOLOv8 model.

A depth estimation module calculates the spatial distance between detected objects, enabling accurate judgment of whether the bat or foot crossed the crease before the wicket was broken. The system operates in real time and demonstrates practical application of deep learning in sports analytics.

Student Printing Service Web Application

Web Development • Cloud Deployment • Production System

This web application allows college students to upload documents remotely for printing and collect the printouts using a system-generated order ID. Students no longer need to physically visit the printing center multiple times to submit files.

The platform is currently live and actively used by more than 400 students every day. It improves operational efficiency, reduces waiting time, and provides better accessibility across the campus.

Smart Geo-Fenced Safety System for Licensed Firearms

IoT • Embedded Systems • Hardware Prototyping

This project focuses on improving firearm safety using an IoT-based access control mechanism. An ESP32 microcontroller, RFID authentication, and a servo motor work together to automatically lock or unlock the trigger mechanism based on authorized access.

A physical prototype was designed and safely tested using a 3D-printed PLA model created in Fusion 360. The system demonstrates secure embedded design and hardware-software integration.

Smart Disease Prediction from Medical Reports

OCR • Machine Learning • Healthcare Automation

This web application extracts text from medical reports including handwritten, scanned, and typed documents using Optical Character Recognition (OCR). The extracted data is processed using a trained machine learning model to predict potential diseases.

The platform helps hospital staff reduce manual workload, improve diagnosis speed, and assign doctors more efficiently in high-traffic medical environments.

Ration Shop Management & Automated Citizen Notification System

Smart Governance • Messaging Automation • Social Impact

This system enables ration shop managers to update shop timings, product availability, and holiday schedules through a secure web portal. Citizens receive automated notifications via SMS, WhatsApp, or voice calls, even without smartphones or internet access.

The platform reduces unnecessary travel, improves transparency, and ensures better accessibility for rural communities. It also maintains collection tracking and reporting for operational monitoring.

Instagram AI Lead Capture & Chat Automation

RAG • CRM Automation • Production AI Systems

An enterprise-grade automation system that intelligently reads Instagram messages, responds automatically, and captures lead details such as name, phone number, and email ID. Conversation memory is maintained using a Retrieval-Augmented Generation (RAG) pipeline backed by Qdrant.

Sentiment analysis identifies high-potential customers. Qualified leads are automatically synced into Google Sheets and Zoho CRM for the marketing team to process, while spam and duplicates are filtered.

AI Influencer Marketing Agent

Automation • AI Video Generation • Scalable Systems

This automation platform converts influencer photos and scripts into professional social media videos without manual editing. The pipeline handles background replacement, AI voice generation, and video rendering automatically.

The system supports high-volume content generation for influencer marketing campaigns and significantly reduces production cost and turnaround time.
